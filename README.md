# Вадос Сервис — сайт автосервиса (онлайн-запись)

Современный адаптивный одностраничный сайт автосервиса в Саратове.
Клиент оставляет марку, модель и симптомы — **заявка сразу приходит менеджеру в Telegram**.

Чистый HTML + CSS + JavaScript, без сборки и зависимостей. Готов к публикации на **Vercel** или **Netlify**.

---

## 📁 Структура

```
vados-service/
├── index.html        # разметка страницы (SEO-friendly)
├── styles.css        # стили, адаптив, анимации
├── script.js         # форма, выбор марки/модели, отправка в Telegram
├── vercel.json       # конфиг для Vercel
├── netlify.toml      # конфиг для Netlify
├── robots.txt
├── sitemap.xml
└── assets/img/       # сюда положите hero.jpg (фон главного экрана)
```

## 🔑 1. Подключение Telegram-бота (ОБЯЗАТЕЛЬНО)

Откройте файл **`script.js`** — в самом верху есть две строки:

```js
const BOT_TOKEN = "ВСТАВЬТЕ_ТОКЕН_БОТА";   // ← сюда токен
const CHAT_ID   = "ВСТАВЬТЕ_CHAT_ID";       // ← сюда chat id
```

**Где взять значения:**

1. **BOT_TOKEN** — напишите в Telegram боту **@BotFather** → `/newbot` → получите токен вида
   `123456789:AAH...`.
2. **CHAT_ID** — напишите своему новому боту любое сообщение, затем откройте в браузере:
   ```
   https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
   ```
   Найдите `"chat":{"id": 123456789}` — это и есть ваш CHAT_ID.
   Для отправки в группу: добавьте бота в группу, ID группы будет с минусом (например `-1001234567890`).

После вставки значений форма будет отправлять заявки в Telegram.

## 🖼 2. Фон главного экрана

Положите фото в `assets/img/hero.jpg` (автомобиль или мастерская/СТО).
Если файла нет — автоматически покажется тёмный градиент, сайт не сломается.

### Фото услуг (по желанию)
Сейчас на карточках услуг — стилизованные плашки. Чтобы поставить реальные фото,
в `styles.css` найдите `.service-img` и задайте картинкам `background-image`
(можно добавить классы для каждой услуги). Скажите, если нужно — настрою под ваши файлы.

## 🚀 3. Публикация

### Вариант А — Netlify (проще всего)
1. Зайдите на https://app.netlify.com → **Add new site** → **Deploy manually**.
2. Перетащите папку `vados-service` в окно — сайт опубликуется.
   (или подключите GitHub-репозиторий — Netlify сам подхватит `netlify.toml`).

### Вариант Б — Vercel
1. Залейте папку в репозиторий GitHub.
2. На https://vercel.com → **Add New → Project** → выберите репозиторий → **Deploy**.
   Настройки билда не нужны, всё в `vercel.json`.

---

## 🔒 Безопасный вариант (опционально)

При прямой отправке из браузера токен бота виден в коде страницы.
Для лендинга это допустимо. Если хотите спрятать токен — вынесите отправку
в serverless-функцию:

- **Vercel:** создайте `api/lead.js`, токен храните в переменной окружения,
  а в `script.js` шлите запрос на `/api/lead` вместо `api.telegram.org`.
- **Netlify:** аналогично через `netlify/functions/lead.js`.

Скажите — добавлю готовую функцию.
